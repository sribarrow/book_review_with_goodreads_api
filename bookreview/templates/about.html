About this project

This is a project work for CS50's Web Programming with Python and JavaScript run by edx as part of the HarvardX: CS50W program.

The technologies used here are Python based modules such as Flask, SQL Alchemy (ORM), FORMS, CSRF implementing a book review site and to provide an API for users to extract information from this website in JSON format for reuse.
